Visual Studio Code: El Centro de Mando

Configuración de Entornos de Desarrollo Modernos

1. Más que un Editor de Texto

Diferencia entre Bloc de Notas, Word y VS Code.

¿Por qué VS Code?

  • Ligero: Funciona en casi cualquier laptop.
  • Extensible: Se adapta a Arduino, Python, C++, etc.
  • Open Source: Respaldado por una comunidad gigante.

2. Estética y Productividad

Un programador pasa 8h al día aquí. Hagámoslo cómodo.

Tema Dracula & Icons

Instalaremos la "Identidad Geek":

  • Dracula Official: Alto contraste, fatiga visual mínima.
  • Material Icon Theme: Reconoce archivos por icono (.ino, .stl, .html).

Settings JSON

Configuraremos el "Format on Save":


{
  "editor.formatOnSave": true,
  "editor.fontSize": 16,
  "workbench.iconTheme": "material-icon-theme",
  "workbench.colorTheme": "Dracula"
}

3. Súper Poderes

Extensiones que usaremos en este taller:

El Kit de Supervivencia

  • Markdown All in One: Para tus bitácoras.
  • Prettier: Para que tu código siempre esté ordenado.
  • Live Preview: Para ver cambios en HTML sin salir del editor.

Taller Práctico

  1. Instalar VS Code (User Installer).
  2. Aplicar el tema Dracula y Material Icons.
  3. Instalar Prettier y activar "Format on Save".
  4. Crear tu primera carpeta de proyecto: Proyectos_TECBA.
