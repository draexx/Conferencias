Visual Studio Code (IDE)

Configuración del "Centro de Mando" Digital

¿Por qué VS Code?

  • Es el estándar de la industria.
  • Multilenguaje (Arduino, Python, JS, Markdown).
  • Ecosistema de extensiones infinito.

1. Estética y Legibilidad

Instalación desde el Marketplace:

  • Tema: Dracula Official
  • Iconos: Material Icon Theme

Tip: Los iconos nos ayudan a identificar archivos .ino (Arduino) y .stl (3D) de un vistazo.

2. Extensiones de Supervivencia

  1. Markdown All in One: Para bitácoras técnicas.
  2. Prettier: Formateo automático de código.
  3. Live Preview: Ver cambios en tiempo real.

3. Configuración Pro

Presiona Ctrl + , y busca:


"editor.formatOnSave": true,
"editor.fontSize": 16,
"workbench.iconTheme": "material-icon-theme"

Actividad Práctica

1. Instalar VS Code.

2. Configurar el tema y los iconos.

3. Crear la carpeta raíz: Proyectos_TECBA.
