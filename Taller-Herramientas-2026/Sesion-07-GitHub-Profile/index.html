GitHub como Identidad Profesional

1. De Local a Global

El concepto de Push.

¿Qué es GitHub?

  • Respaldo: Tus proyectos están seguros ante fallos de hardware.
  • Colaboración: La base del Open Source.
  • Currículum: Tus contribuciones valen más que un papel.

2. El Profile README

Tu carta de presentación al mundo.

El Repositorio Secreto

Crear un repo con el mismo nombre que tu usuario.

Ejemplo: github.com/pedro-carranza/pedro-carranza

¿Qué incluir?

  • Biografía: Quién eres y qué estás aprendiendo.
  • Tech Stack: Iconos de las herramientas que manejas.
  • Stats: Tus estadísticas de GitHub (GitHub Readme Stats).

3. Beneficios Estudiantiles

GitHub Student Developer Pack

Usa tu correo @tecba.edu.bo para obtener:

  • Copilot (IA para programar) gratis.
  • Créditos en nubes (Azure, AWS, DigitalOcean).
  • Dominios .me o .tech gratuitos.

Taller Práctico

  1. Hacer Publish de tu repositorio local a GitHub.com.
  2. Crear tu Special Repository (Username Profile).
  3. Diseñar tu biografía usando el Markdown aprendido en la Sesión 4.
  4. Solicitar el Student Developer Pack con el correo institucional.
