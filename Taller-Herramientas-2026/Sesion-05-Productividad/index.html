Shortcuts, Formatters y Snippets

1. Atajos de Teclado (Shortcuts)

Si usas el mouse, estás perdiendo tiempo.

Navegación Veloz

  • Ctrl + P : Buscar cualquier archivo.
  • Ctrl + Shift + P : Paleta de comandos (¡Todo está aquí!).
  • Ctrl + B : Mostrar/Ocultar barra lateral.

Edición Multilínea

  • Alt + ↑ / ↓ : Mover línea actual.
  • Shift + Alt + ↑ / ↓ : Duplicar línea.
  • Ctrl + D : Seleccionar siguiente coincidencia.

2. El Poder de los Formatters

Deja que VS Code limpie tu código por ti.

Prettier en Acción

¿Recuerdan el "Format on Save" de la Sesión 3?

  • Indentación automática.
  • Corrección de comillas y puntos y coma.
  • Tablas de Markdown perfectamente alineadas.

Extensiones "Cosméticas"

  • Indent-rainbow: Colorea la indentación.
  • Color Highlight: Muestra colores reales en el código CSS/Hex.
  • Better Comments: Comentarios con alertas // ! TODO.
